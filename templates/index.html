Live demo • browser MediaPipe + server model

ASL Alphabet — SignWriter

This version runs hand-landmarks in your browser (MediaPipe JS) and sends a 63-number vector to POST /predict on the server, which uses your model_svm.pkl.

—

Notes:
- You’ll be asked for camera permission.
- The model runs on the server; landmarks run in the browser.
- Endpoint: POST /predict

Want the original WebSocket version? Run it locally from the repo root: 

pip install -r requirements.txt
python app.py
